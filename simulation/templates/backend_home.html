API backend

Restaurant queue optimization

This host runs the Django REST service for simulations, restaurants, and auth. Use the paths below or integrate from the web app.

Quick links

  • GET /api/health/ Liveness & version
  • POST /api/auth/login/ JWT obtain
  • POST /api/auth/register/ Create account
  • GET /admin/ Django admin

Simulations & restaurants

  • POST /api/simulation/run/ Authenticated run
  • POST /api/simulation/run-guest/ Guest run
  • GET /api/simulations/ List saved runs
  • GET /api/restaurants/ Restaurants CRUD (see API docs)

JSON APIs expect Content-Type: application/json unless noted otherwise.

Queue Optimization System: backend only. Pair with the Next.js frontend for the full UI.
